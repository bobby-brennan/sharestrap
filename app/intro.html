Strapping!

This is a demo of Strapping, an in-browser Bootstrap editor You can incorporate Strapping into your project in two ways:
  1. Include Strapping in development mode to help you customize your site's styles
  2. Include Strapping in production to allow your users, customers, and teams create themes for your site
Here's an example: 

<script src="strapping.min.js"></script>
<script>
<body onload="strapping.initialize()">
</body>
</script>
This will load Strapping in a fixed-position side column (like you can see on the left now). There can edit your bootstrap variables. Clicking Set Styles will re-compile bootstrap and apply the styles to the page. You can listen for changes as well: strapping.onCompiled(function(result) { console.log(result.status); // 0 if everything went OK console.log(result.message); // Check this if status !== 0 console.log(result.variables); // A JavaScript object mapping variables to values console.log(result.css); // Full Bootstrap CSS console.log(result.scss); // A _variables.scss file for inclusion in Bootstrap });
